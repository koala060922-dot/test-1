### 순서 목록

1. 번호를 붙여서 사용
2. 엔터를 눌러 줄 바꿈


### 순서 없는 목록

+ +와 - 또는 *를 붙여서 나열
- 기호와 내용은 띄어쓰기
* 기호를 구별하지 않음

#### 텍스트 꾸미기

**굵게**는 **과 __를 앞뒤로 사용
*기울임체*는 *와 _를 앞뒤로 사용
~취소선~은 ~를 앞뒤로 사용
***굵은 기울임체***를 ***와 ___를 앞뒤로 사용

##### 소스코드 입력하기

여러 줄 짜리 소스코드
```Javascript
<script>
 let now = new Date();
 let display = now.toLocaleTimeString();

</script>
```
